<objective>
Edit 3 lab guide markdown files to (1) flatten the student workspace from `llmops-project/lab-01/` sub-directories to a single flat `llmops-project/` directory, and (2) replace student-facing `pip install` commands with `uv pip install --system`.

Purpose: The per-lab sub-directory pattern in lab guides was inconsistent with the K8s layer (which already uses flat `/mnt/project/` paths). This caused students to put files in the wrong location, breaking the training Job. Switching to `uv` gives 10-100x faster installs (relevant for sentence-transformers + PyTorch).

Output: 3 updated lab markdown files with consistent flat workspace paths and uv commands.
