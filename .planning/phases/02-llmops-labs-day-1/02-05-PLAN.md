    - `@cl.on_message async def on_message(message: cl.Message)`:
      * **Step 1 — RAG Retrieval** (`cl.Step(name="Retrieving clinic documents", type="tool", default_open=False)`):
        - `s.input = message.content`
        - POST to `{RETRIEVER_URL}/search` with `{"query": message.content, "k": TOP_K}`

      * **Step 2 — Prompt Construction** (`cl.Step(name="Building prompt", type="run", show_input="json")`):

        - `s.input = messages` (shown as collapsible JSON in UI)

      * **Step 3 — LLM Generation** (`cl.Step(name="LLM generation", type="run")`):

        - `s.output = "Generation complete"`

      * Wrap retriever and vLLM calls in try/except; on error send error Message to user
